# GitHub Upload Instructions

Use these files in the public GitHub repository:

MoroccoLocalSupport / morocco-local-support

## Option 1: Upload files manually

1. Open the repository on GitHub.
2. Click **Add file**.
3. Click **Upload files**.
4. Unzip this ZIP file on your computer.
5. Drag all `.md` files into GitHub.
6. Add a commit message such as:
   `Add Morocco Local Support documentation files`
7. Click **Commit changes**.

## Option 2: Create files one by one

1. Click **Add file**.
2. Click **Create new file**.
3. Copy the file name exactly.
4. Paste the content.
5. Commit changes.
